03. Projects

AI / RAG

TMPPM Policy Indexer

AI agent that delivers citation-backed answers to any Texas Medicaid PA question in seconds.

10-20 min → 60 sec Policy search time
900+ pages Indexed by RAG agent

Problem: The Texas Medicaid Provider Procedures Manual is 900+ pages of dense regulatory and clinical policy. PA clinicians and supervisors routinely spend 10–20 minutes searching for a single policy clarification, and still risk misinterpreting or missing relevant sections.

What I built: A Microsoft Copilot agent using retrieval-augmented generation (RAG) that indexes the full TMPPM corpus and returns structured, citation-backed answers. Every response includes a confidence score calibrated to a defined rubric, source citations with document title, section number, and page number, applicable conditions and exceptions, risk flags for edge cases, and related manual sections. The agent distinguishes between 'must,' 'may,' and 'should' per manual language, escalates when confidence drops below threshold, and explicitly flags when a question falls outside TMPPM scope.

Microsoft Copilot Studio RAG Prompt Engineering Knowledge Management

Full-Stack / Algorithm

Operations Dashboard + Planner

Flask dashboard with real-time inventory visibility and a greedy algorithm that optimizes daily assignments.

15-20 min saved Daily planning per supervisor

Problem: Supervisors had no real-time visibility into authorization queue status. Daily work assignment was done by gut feel — eyeballing inventory and assigning staff without structured methodology. This led to imbalanced workloads and missed SLA deadlines.

What I built: A Flask web app with two modules. The dashboard pulls live inventory counts and displays them in bar charts grouped by operational area. The daily plan generator takes current inventory, available staff, shift hours, and area qualifications as inputs, then outputs optimized assignments using a greedy algorithm that prioritizes: (1) inventory due today, (2) areas only workable by few qualified staff, (3) highest remaining volume.

Python Flask Chart.js Greedy Algorithm
